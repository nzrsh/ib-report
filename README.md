**IB Report** — веб-приложение для регистрации и учёта событий и инцидентов информационной безопасности.

Основные возможности:

- Создание отчётов о событиях ИБ и связанных инцидентах через многошаговую форму
- Разделение на события и полноценные инциденты с классификацией
- Авторизация пользователей (роли admin/user)
- Таблицы событий и инцидентов с фильтрацией и пагинацией
- Генерация PDF-отчётов по событиям и инцидентам
- Хранение данных в SQLite через Prisma ORM
- Бэкенд на NodeJS, фронтенд на чистом HTML + vanilla JS

Стек технологий:

- Backend: NodeJS, Prisma, SQLite
- Frontend: HTML, CSS, vanilla JavaScript
- PDF-генерация: pdfkit
- Аутентификация: JWT
